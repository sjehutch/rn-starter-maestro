# RN Starter

A tiny, strict Expo + TypeScript starter with file-based routing.

## Quick start

```sh
npm install
npx expo start
```

Open platforms from the Expo CLI:
- Press `i` for iOS Simulator
- Press `a` for Android Emulator

## Common commands

```sh
npm run start
npm run start:clear
npm run typecheck
npm run lint
```
